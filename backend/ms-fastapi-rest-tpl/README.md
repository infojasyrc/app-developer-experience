## Getting started

### Requirements

Install a container tech: Docker

```bash
brew install --cask docker-desktop
```

### Local Development

```bash
# run all commands available
make
# build dev container for local
make build-dev
# install dependencies
make install-dependencies
# get inside the container
make interactive
# run all unit tests
make run-tests
```
